Spam Link Detector

Protect yourself from phishing. Learn how to create a powerful automation that instantly analyzes copied links and warns you of suspicious patterns.

Quick Start Summary

This automation runs locally on your device. It triggers when clipboard content changes, checks against a regex pattern, and alerts you.

schedule 5 mins signal_cellular_alt Beginner
checklist

Prerequisites

  • check iPhone running iOS 15 or later
  • check Shortcuts App (Pre-installed)
  • check No coding experience required
1
1

Open Automation Tab

Start by opening the Shortcuts app and navigating to the Automation tab at the bottom of the screen.

Action Steps

  • chevron_right Open the Shortcuts app on your iPhone.
  • chevron_right Tap the Automation tab at the bottom.
  • chevron_right Tap the + button in the top-right corner.
2
2

Create Personal Automation

You'll see the automation setup screen. This is where you define the trigger (When) and the actions (Do).

Action Steps

  • chevron_right Select Create Personal Automation.
  • chevron_right You'll see the "When" and "Do" sections ready to configure.
info

Note: Ensure "Run Immediately" is selected so the automation executes without manual confirmation.

3
3

Configure Message Trigger

Set up the trigger to activate when you receive a message containing "http" - the start of any web link.

Action Steps

  • chevron_right Tap the "When" section and search for Message.
  • chevron_right Set Sender to "Any Sender".
  • chevron_right Set Message Contains to http.
  • chevron_right Enable Run Immediately.
4
4

Add Regex Pattern

This is the brain of the operation. The regex pattern identifies suspicious URL structures commonly used in spam and phishing.

regex-pattern.txt
(http://(?!2[^\.]10\.|192\.168\.|172\.(1[6-9]|2[0-9]|3[01])\.)[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3})|(\.(?:xyz|top|zip|gq|ml|ga|cf|tk|pw|link|work|click|loan|download|racing|review|trade|webcam|bid|win|stream|party|cricket|cu|buzz|live|online|site|club)|(\W|\$))|(sbi|hdfc|icici|axis|pnb|bob|kotak|paytm|phonepe|gpay)[\-\.]?\w*\.(xyz|top|link|click|online|site|info)|(gov(t)?[\-]?(in|india)?)\.(xyz|top|link|click|online)|(verify|update|suspended|blocked|urgent|claim|refund|lottery|winner)[\-]?\w*\.(xyz|top|link|click)|({2,}|{2,})|(bit\.do|cutt\.ly|short\.io|tiny\.cc|shorturl\.at)

Action Steps

  • chevron_right In the "Do" section, add Receive messages as input.
  • chevron_right Add the Match action.
  • chevron_right Paste the regex pattern above into the Match field.
5
5

Set Up Alert Logic

Define what happens when a suspicious link is detected. The automation will show a warning alert to protect you from clicking.

Action Steps

  • chevron_right Add an If condition: "If Matches has any value".
  • chevron_right Inside the If block, add Show Alert.
  • chevron_right Customize the alert message: Spam LINK from [Sender] don't click any links.
  • chevron_right Add Otherwise and End If to complete the logic.
check_circle

You're Done! Your automation will now scan incoming messages and alert you when suspicious links are detected.
